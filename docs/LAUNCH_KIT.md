`holochain-ballot` is an experimental voting hApp on Holochain with elections, voter registration, ballots, and transparent tallies.  
It includes a static showcase and a live Tryorama dashboard so people can see both the concept and real DNA behavior.  
It is not a certified public-election system, and secret-ballot DNA is explicitly NO-GO in this repo scope.

## Longer launch post

I just open-sourced **holochain-ballot**: an experimental voting hApp built on Holochain.

What it includes:
- Election creation, voter registration, vote casting, and tallying
- Transparent tally path with documented audit/recount notes
- A static showcase UI for explaining mechanics
- A live dashboard that streams a real Tryorama run

Scope is intentional: this is for learning, demos, and controlled org/community pilots where participants accept the transparent trust model. It is not positioned as a certified public-election system. Secret-ballot/high-stakes implementation is documented as NO-GO in this repo.

If you work on Holochain governance flows, pilot design, or auditability patterns, feedback is welcome.

Repo: https://github.com/ta10101/holochain-ballot
