holochain-ballot — live DNA demo

Same machinery as npm run docker:test, with a visual timeline. The run creates a draft election, then the creator calls advance_election_phase before registration and again before voting (see docs/ELECTION_LIFECYCLE.md). Four players default to Alex, Blake, Casey, Dana (same names as the Vite showcase lab). The right column is a raw-style log of what Tryorama is doing.

Not for government or certified public elections—developer education and pilot-style demos only. See docs/SCOPE.md in the repo.
